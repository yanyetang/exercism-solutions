# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Structure

Exercises are organized by language: `bash/` and `python/`. Each exercise lives in its own subdirectory with the solution file(s) and auto-generated test file.

## Running Tests

**Python** (from `python/` directory):
```bash
uv run pytest leap/                  # run a single exercise
uv run pytest                        # run all python exercises
```

**Bash** (from within the exercise directory):
```bash
bats hello_world.bats                # run a single exercise
```

## Syncing with GitHub

After completing an exercise:
```bash
git add .
git commit -m "Complete <exercise-name> in <language>"
git push
```

## Downloading New Exercises

```bash
exercism download --exercise=<name> --track=python
exercism download --exercise=<name> --track=bash
```

## Exercise File Conventions

- **Python**: solution in `<exercise>/<exercise>.py`, tests in `<exercise>/<exercise>_test.py`
- **Bash**: solution in `<exercise>/<exercise>.sh`, tests in `<exercise>/<exercise>.bats` (requires `bats-extra.bash` helper — do not delete it)
- `.exercism/` metadata folders are auto-generated by the CLI — do not edit
